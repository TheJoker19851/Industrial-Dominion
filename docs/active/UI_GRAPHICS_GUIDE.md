# UI_GRAPHICS_GUIDE.md

## Visual philosophy

A modern industrial command center / financial strategy terminal.

## Core screens

- Dashboard
- Market
- Buildings
- Region map
- Production chains
- Messaging
- Corporation chat
- News feed
- Settings / language

## Mobile UI notes

- use bottom navigation or compact navigation on phone
- use drawers/sheets instead of permanent sidebars
- convert dense tables to cards or stacked rows on narrow screens
- core actions must stay thumb-friendly
- avoid requiring hover interactions
