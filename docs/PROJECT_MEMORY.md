# PROJECT_MEMORY.md

## Identity

Industrial Dominion is a browser-based economic MMO focused on industry, trade, speculation, corporations, and market power.

## Core decisions already made

- Web-first, no app-store dependency
- Minimal graphics, strong interface
- Player-driven economy is the heart of the game
- Persistent world preferred over hard resets
- Corporations included before alliances
- Alliances are planned for V2+
- Communication is essential:
  - private messaging
  - corporation chat
  - system news feed
  - forum later
- Monetization must remain fair and non-pay-to-win
- Individual disasters are removed from the design permanently
- Only global and regional economic events exist
- Multilingual support is a major requirement
- English and French are supported from the beginning
- Backend stack is Fastify + Railway + Supabase
- Frontend stack is Vercel-hosted React/Vite
- Mobile support is required for V1 core loops
