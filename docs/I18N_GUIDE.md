# I18N_GUIDE.md

## Launch languages

- English (en)
- French (fr)

## Mandatory rules

1. All UI text uses translation keys.
2. Content entities use canonical IDs + translation keys.
3. Use locale-aware formatting for currency, numbers, percentages, and dates.
4. News feed text must be localizable.
5. Errors, empty states, onboarding text, and billing text must be translated.
