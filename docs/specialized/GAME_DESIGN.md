# GAME_DESIGN.md

## Core fantasy

Build an industrial empire in a living multiplayer economy.

## V1 social

- private messaging
- corporation chat
- system news feed

## V1 exclusions

- no alliances
- no individual disasters

## V1 events

- global economic events
- regional economic events
- positive / neutral / negative mix

## V1 device goal

The game must be fully playable on mobile for core loops:

- onboarding
- dashboard
- claim production
- inventory
- simple market actions
- news
- messages
- corporation chat
